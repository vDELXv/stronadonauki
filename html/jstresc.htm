    </script>

ZEGAR

    function zegarek()
    {
      let data = new Date();
      let godzina = data.getHours();
      let minuta = data.getMinutes();
      let sekunda = data.getSeconds();

      if (minuta < 10) minuta = "0" + minuta;
      if (sekunda < 10) sekunda = "0" + sekunda;

      let pokazDate = godzina + ':' + minuta + ':' + sekunda;
      document.getElementById("zegar").innerHTML = pokazDate;

      setTimeout(zegarek, 1000);
    }

-Najpierw deklarujemy zmienną data do której przypisujemy aktualną datę.

-Następnie do nowej zmiennej godzina porzypisujemy godzinę z zmiennej data.

-Następnie do nowej zmiennej minuta porzypisujemy minutę z zmiennej data.

-Następnie do nowej zmiennej sekunda porzypisujemy sekundę z zmiennej data.

-Następnie jeśli minuta jest mniejsza od 10, czyli by była pokazana jako np. 00:8:00 zamiast 00:08:00 to dodajemy 0 przed zmienną minuta.

-Następnie jeśli sekunda jest mniejsza od 10, czyli by była pokazana jako np. 00:00:6 zamiast 00:00:06 to dodajemy 0 przed zmienną sekunda.

-Następnie do nowej zmiennej pokazDate przypisujemy zmienną godzina i dodajemy dwukropek w cudzysłowie itd. aż do zmiennej sekunda.

-Następnie w elemencie HTML o id zegar wczytujemy zmienną pokazDate.

-Następnie wywołujemy funkcję setTimeout(zegarek, 1000), która służy do wywoływania z opóźnieniem funkcji w pierwszym parametrze (zekarek). W drugim parametrze podajemy czas w milisekundach po jakim zostanie ta funkcja wywołana. Czyli w tym przypadku co 1s będzie się odświeżać data.

-W body wpisujemy onload="zegarek()" co oznacza że funkcja zegarek() wykona się natychmiast po załadowaniu strony i będzie się odświeżać funkcaj co 1s (setTimeout).
